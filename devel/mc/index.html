Introduction

I created this framework to support rapid development of Monte Carlo simulations. It can be used in all kinds of numeric analysis. See the README for more detailed information on its use.

Example

Let's say you've been asked to come up with a failure estimate for this scenario:

  • Flash ROM size of 1024 blocks
  • Consists of many segments of varying block sizes
  • 10 to 15 randomly located blocks are bad
  • Flashing fails if any segment has 2 or more bad blocks

An estimate for the failure rate is trivial to get using Monte Carlo. Simply write a function that implements this scenario:

  1. Create an array of 1024 elements
  2. Mark between 10 and 15 elements as bad (using mc_gen_number(10, 15))
  3. Shuffle the 1024 elements
  4. Step through each segment of the array, see if any has 2 or more bad blocks
If a failure occurs, have the function return 1.0. Otherwise, return 0.0. That's it. Plug it into the Monte Carlo framework and see what mu ends up being. This is your average failure rate.
